Daily English Tab

새 탭에서 영어 문장을 가볍게 익히는 확장 프로그램

Daily English Tab은 영어 문장, 한국어 번역, 발음 듣기, 복습 큐, 자동 업데이트를 한곳에 담은 브라우저 확장 프로그램입니다. 복잡한 설정 없이 열자마자 바로 학습을 시작할 수 있도록 설계했습니다.

개인정보처리방침 지원 GitHub Repository

핵심 기능

  • 매일 영어 문장 제공
  • 한국어 번역 표시/숨기기
  • 발음 듣기
  • 복습 목록 저장

자동 업데이트

문장 데이터는 GitHub의 JSON 파일에서 자동으로 갱신됩니다. 사용자는 별도 설정 없이 최신 콘텐츠를 받을 수 있습니다.

신뢰와 개인정보

확장 프로그램은 로컬 저장소만 사용해 학습 상태를 저장합니다. 개인 정보를 수집하거나 외부 서버로 전송하지 않습니다.
